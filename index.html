PORTFOLIO

AFPA RELOADED

J'ai travaillé principalement sur la partie Géolocalroom du projet de stage qui avait pour objectif de simuler une application administrative pour centre de formation.

Dans le cahier des charges : il été demandé de créer une carte interactive du centre de formation, avec une possibilité de réserver les salles pour les formateurs, tout en ayant l'état de réservation des salles en direct, visible comprenant un code couleur proche du logo de l'Afpa.

L'application a pour vocation de garder une traçabilité de ces informations pour l'administration théorique du centre.

Pour une simulation proche de la réalité nous avons donc décidé de nous baser sur les données de notre centre de formation.
