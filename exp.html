Afpa st Jérôme - Marseille (13) 2021

Stage en Développement Web 6 semaines

Dans le cadre de la crise sanitaire, nous avons pu réaliser notre période en entreprise auprès du centre de formation. J'ai contribuer a participer au développement du projet Open Source Afpa_Révolution.

Ce projet avais pour but de développer une application web a but administrative avec un plan avec le status des salles du centre de formation ainsi que la possibilité de réaliser des réservations de salles pour un formateur par éxemple.

J'ai travailler avec Git pour réaliser ce projets au vue du nombre de développeur que nous étions.

J'ai également utilisé principalements les languages de programmation suivant :

  • - HTML/CSS
  • - MySql
  • - PHP
  • - JavaScript
